# EPUB-formater

## Currently support

- Language switch to English, Hebrew, Hindi, Chinese, Korean, Japanese.

## Usage

```shell
# clone repository
git clone https://github.com/soul-powerpoint/EPUB-formater.git
cd EPUB-formater

# build
make
./formater <path_to_your_epub_file> -l <language>

# clean
make clean
```
